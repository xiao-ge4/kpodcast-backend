# 片头背景音乐目录

请将以下背景音乐文件放置在此目录下：

| 文件名 | 风格 | 说明 |
|--------|------|------|
| `bgm_tech.mp3` | 科技 | 科技、创新、AI相关内容 |
| `bgm_business.mp3` | 商业/财经 | 经济、创业、投资相关内容 |
| `bgm_life.mp3` | 生活/日常 | 美食、旅行、生活方式相关内容 |
| `bgm_culture.mp3` | 文化/历史 | 历史、艺术、文学相关内容 |
| `bgm_entertainment.mp3` | 娱乐/轻松 | 电影、音乐、游戏相关内容 |
| `bgm_education.mp3` | 教育/学习 | 学习、知识科普相关内容 |
| `bgm_health.mp3` | 健康/养生 | 医疗、运动、健康相关内容 |
| `bgm_emotion.mp3` | 情感/心理 | 恋爱、人际关系相关内容 |
| `bgm_growth.mp3` | 个人成长 | 自我提升、励志相关内容 |
| `bgm_general.mp3` | 通用 | 通用背景音乐（无语音片头时使用） |

## 注意事项

1. 背景音乐建议时长：15-30秒
2. 格式：MP3
3. 如果某个风格的背景音乐文件不存在，系统会自动回退到旧目录 `assets/bgm/` 中的对应文件
4. 如果都不存在，会使用 `tongyong.MP3` 作为默认背景音乐
